Selector

CSS Selector are used to select the HTML elements and apply styles. Example : element selectro , class selector , id selector.

Box Model

The box model contains content, padding, border, and margin. It controls spacing and size of elements.

Colors

CSS colors can be written using color names, HEX, RGB, and RGBA values.

Flexbox

Flexbox is used to align items in row or column. It is useful for navbar and card layouts.

CSS Grid

CSS Grid is used to create two-dimensional layouts with rows and columns.

Transition

Transition is used to make style changes smooth, like hover zoom effects.

Width and Height

Used to set the size of elements. Example (width:300px; height:200px;)

Background

Used to add background color or image. Example (background: black; background-image:url();)

Typography

Used to style the text in weebsite. Properties that are use { font-size,font-family,font-weight,text-align}

Position

Position are used to control the elemnts and place element where we needed Types of Position they are (static,relative,absolute,fixed,sticky) Example : (position:absolute;)

Transform

Transform used to change the shape size and position of the card in the webpage . functions(scale(),rotate(),translate()) Example : transform:scale(1.2)

Pseudo Class and elements

The Pseudo Class Used to style elements in a specific state and the pseudo element Used to style specific parts of an element. Properties of Pseudo Class : (:hover{} :focus{} :active {} :first-child{})
Properties of Pseudo element : (::before{} ::after{} ::first-letter{} )

Z-index

The Z-index are used to control element layers.Which element should appear front Which element should appear behind Properties : (z-index:10;)
